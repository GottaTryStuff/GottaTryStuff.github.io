    Alembic

    com.unity.formats.alembic 2019.4 verified 

    Description

    Import and export Alembic files (.abc), record and stream animations in the Alembic format directly in Unity.

    The Alembic format is commonly used in animation to transfer facial, cloth, and other simulation between applications.

    IMPORTANT: Alembic for Unity only supports Windows 64-bit, MacOS X and Linux 64-bit as build targets.

    Verified for Unity
